Box Shadow : This allows to create a drop shadow for an element.

Gradient : this is for smooth transitions between specified colors. It has two types; linear gradients and radial gradient.

New Selectors : for choosing DOM elements based on the attributes. It allows styling by utilizing the specific attribute. It can match by value, prefix and suffix, and substring.

RGBA : this is given an additional feature to control opacity of elements.
