Does it matter to use relative size on styling ?

Yes it does. It provides the consistency preferred. Accessibility is not that of a concern considering the browsers capability to adjust. It also modifies the base font-size on the body. And rem is relative to the root element which defines a single font size on the html element and defines all rem units to be a percentage of that.
